# Home Project Designer

A garden shed design tool built with Claude. Features a conversational Q&A flow, rotatable 3D model, comprehensive materials takeoff with Good/Better/Best tier selection, cost estimation, and a printable Home Depot shopping list.

## Files

- `Home Project Designer.html` — Main entry point
- `src/app.jsx` — Main app component
- `src/questionFlow.jsx` — Question definitions
- `src/shedModel.jsx` — 3D/plan view rendering
- `src/panels.jsx` — Cost, practices, suggestions panels
- `src/materialsCatalog.jsx` — Materials catalog with tier pricing
- `src/materialsPanel.jsx` — Materials UI panel
- `src/printPlan.jsx` — Printable build plan
- `src/shoppingList.jsx` — Printable Home Depot shopping list
